### php-cs-fixer

### phpstan
Error level [0..9] (default = 0), currently using level max (9)
```sh
 ./tools/phpstan/vendor/bin/phpstan analyse src --level=max
 ./tools/phpstan/vendor/bin/phpstan analyse domain --level=max
```
